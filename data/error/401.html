Forgotten Password

If you have forgotten your password, anyone else who has a conductor password or who is logged in as a conductor can reset it for you.

SSH

If no one is able to reset your password, you can use an ssh connection to reset passwords.

  • On another computer, make an ssh connection to ssh pi@immrama.local.
    • If your computer can't find immrama.local:
      • Make sure your computer is connected to the immrama WiFi network and Try again.
      • If that doesn't help, try pi@immrama.home, pi@immrama.localnet, pi@172.24.1.1, and pi@immrama
    • If you cannot remember the ssh password, consult the written documentation to find the default password.
  • Type: sudo cp /boot/htpasswd/lighttpd-htdigest.default /boot/htpasswd/lighttpd-htdigest.user
  • Type: sudo service lighttpd restart
  • Type: exit
  • Your ssh connection should now be over.
  • You can now connect with username: conductor password: defaultpass
  • Please change your password.

SD Card

You can also change your password via an SD card reader. This method requires crashing the computer by pulling the plug witout first doing a proper shutdown. Ocassionally, doing this may corrupt the SD card and so it should be avoided as muh as possible.

  • If the computer is on, pull the plug
  • Take out the SD card
  • Using another computer with an SD card reader, go to the /boot partition.

  • There will be a folder in there called 'htpasswd'
  • In the htpasswd folder, throw away the file '/boot/htpasswd/lighttpd-htdigest.user'
  • Make a new copy of '/boot/htpasswd/lighttpd-htdigest.default' called /boot/htpasswd/lighttpd-htdigest.user
  • Properly unmount the SC card.
  • Return the SD card to the Raspberry Pi computer
  • Plug in the computer to boot it.
  • You can now connect with username: conductor password: defaultpass
  • Please change your password.
